# Squad Routing

## Rules

| Signal | Agent | Reason |
|--------|-------|--------|
| WPF / XAML / C# implementation, UI layout, controls | Fenster | WPF Dev — owns all implementation |
| Architecture decisions, code review, scope | Keaton | Lead — owns design and review |
| Tests, edge cases, quality, regression | Hockney | Tester — owns quality |
| Logging, decisions merge, session summaries | Scribe | Silent logger |
| Backlog, GitHub issues, work queue | Ralph | Work monitor |
| Multi-domain (UI + backend + tests) | Keaton + Fenster + Hockney | Parallel fan-out |
